一、需要具备的东西

2.一台思科2911/K9

3.网线

激活步骤

1.PC开启TFTP服务

2。将license拷到路由器中

router#copy tftp flash0:

按提示输入tftp地址、文件名。然后确认即可

3.安装

router#licence install flash0:xxxxxxx.lic

4.安装完成，reload之后show version
