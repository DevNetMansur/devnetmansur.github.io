NFS：Network File System 网络文件系统，可以让电脑在本地直接管理远端的硬盘。依赖RPC.
RPC：Remote Procedure Call 远程过程调用，计算机通信协议。该协议允许运行于一台计算机的程序调用另一台计算机的子程序，而程序员无需额外地为这个交互作用编程。
